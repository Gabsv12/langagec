# Morpion en C avec ncurses

Ce projet implémente un jeu de morpion en ligne de commande avec une interface interactive grâce à la bibliothèque ncurses.

## Fonctionnalités

- Interface terminal interactive
- Joueur vs Joueur, Joueur vs Machine, Machine vs Machine
- Stratégie de base pour le bot
- Navigation clavier avec flèches (ou touches associées)
- Compatible Linux (testé sur Kali)

## Compilation

Utiliser CLion (ou manuellement avec `cmake . && make`)

## Exécution

```bash
./tictactoe
